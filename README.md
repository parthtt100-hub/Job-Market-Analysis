# End-to-End Job Market Analysis Dashboard

## Project Overview
This project is an end-to-end Data Analysis project built using Excel, Python, MySQL, and Power BI.  
The objective of this project is to analyze job market trends, salary patterns, hiring locations, industries, and top-paying companies using real-world job listing data.

The project follows the complete Data Analytics workflow:
- Data Cleaning
- Data Transformation
- SQL Analysis
- Data Visualization
- Dashboard Development

---

## Tools & Technologies Used

| Tool | Purpose |
|---|---|
| Excel | Initial data understanding and preprocessing |
| Python (Pandas) | Data cleaning and transformation |
| MySQL | Data querying and analysis |
| Power BI | Interactive dashboard and visualization |

---

## Dataset Information

The dataset contains job listings with information such as:
- Job Title
- Company Name
- Salary Estimate
- Industry
- Location
- Company Ratings
- Company Details

---

## Project Workflow

### 1. Data Cleaning (Excel + Python)
- Removed duplicate records
- Handled missing values
- Cleaned salary-related columns
- Standardized data formats
- Exported cleaned dataset for analysis

### 2. SQL Analysis (MySQL)
Performed analytical queries such as:
- Jobs by location
- Jobs by industry
- Average salary by role
- Top paying companies
- Salary distribution analysis

### 3. Power BI Dashboard
Created an interactive dashboard containing:
- KPI Cards
- Salary analysis
- Hiring distribution
- Industry insights
- Top paying companies
- Interactive slicers and filters

---

## Key Insights
- Identified locations with highest hiring activity
- Analyzed highest paying job roles
- Compared salary distribution across industries
- Found companies offering higher salary packages
- Visualized job market insights using Power BI
